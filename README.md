# Bottleneck and Pedestrian Traffic Flow Analysis in Bulacan State University, Malolos Bulacan Campus
### Mathematical Modeling FCO - BSMCS 3A
# Overview
This study applies graph theory to model and analyze the pedestrian within Bulacan State University- Malolos Campus. The graphs represented as campus pathways. To analyze the campus shortest path and bottleneck was calculated using computational tool.
### 1. Install Dependencies
```bash
python -m pip install -r requirements.txt
```
### 2. Run
```bash
python main.py
```
# Output
| File | Description |
| :--- | :--- |
| `.sage` | SageMath code for BulSu network.|
|`pedestrian network`| Campus nodes and Edges.|
|`Shortest_distance_to_FH`| Source node to FH.|
|`Shortest_path_Congestion`| Consedering the capacity of pathway.|
